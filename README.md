# 💼 CV Digital - Frontend Developer

Currículum vitae interactivo y moderno construido con React y Vite. Presenta experiencia profesional, proyectos destacados y datos de contacto.


---

## ✨ Características

- 👤 **Perfil profesional** - Foto y datos de contacto
- 📚 **Experiencia y formación** - Detalle de trayectoria profesional
- 🚀 **Proyectos destacados** - 4+ proyectos con descripción y links
- 📧 **Enlaces de contacto** - Email, LinkedIn y GitHub
- 📱 **Diseño responsivo** - Adaptado a todos los dispositivos

---

## 🛠️ Tecnologías

- **React 19.2.4** - Interfaz de usuario
- **Vite 8.0.0** - Build tool
- **CSS3** - Estilos y responsive design
- **ESLint** - Calidad de código

---

## 🚀 Instalación

```bash
# Clonar
git clone https://github.com/tu-usuario/CV-WEB.git
cd CV-WEB-main

# Instalar
npm install

# Desarrollar
npm run dev

# Producción
npm run build
```

---

## 📁 Estructura

```
src/
```
